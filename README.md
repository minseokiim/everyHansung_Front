## 🖥️ 프로젝트 개요

<br/>
한성대학교 커뮤니티 서비스
<br/>
<br/>

## 🔎 프로젝트 소개

<br/>


한성대학교 학생들을 위한 커뮤니티 서비스<br/>
- 학교 생활 정보, 익명 커뮤니티, 챗봇, 시간표 관리, 쪽지, 책방 등의 기능을 제공 <br/>
- 챗봇에 자주 묻는 질문, 간단한 정보들에 대한 답을 학습시켜 편하게 이용 가능
<br/>
